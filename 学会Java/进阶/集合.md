#### Collection遍历

##### 迭代器

```java
Iterator<Student> itr = c.iterator();

while(itr.hasNext()) {
  Student s = itr.next();
  System.out.println(s);
}
```



##### 增强for循环

底层也是迭代器

```java
for (Student student : c) {
           //  
        }
```

##### foreach

底层也是迭代器

```java
c.forEach(new Consumer<Student>() {
            @Override
            public void accept(Student student) {
                System.out.println(student);
            }
        });
```
